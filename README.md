# csc125-final-project
## Project Description
This program will be a text based game where the player is venturing through a cave and runs
into a monster. The player has the option to either fight the monster, or retreat out of the cave.
## Features
- Dictionary storing Health Values for the player and monster, and money for player
- Custom function `battle_sequence` that takes input of player and monster HP, and outputs who won the battle
- while loop that waits for the user to decide whether to fight or retreat.
- User input with validation
- Formatted output
